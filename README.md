# Work hours
Work hours evidence
## Requirements
* Visual Studio 2017 +
* MS SQL Server 2008 +
## Installation
1. Create database on SQL Server with script /SQL/00 - Databasse prepare.sql
2. Create tables in database with all scripts in /SQL/Tables
3. Create views in database with all scripts in /SQL/Views
